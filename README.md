# second-brain

A structured second brain for learning, thinking, long-term knowledge management, and AI-assisted workflows.

## Structure

```text
.
├── knowledge/      # notes
├── templates/      # reusable note templates
├── .obsidian/      # Obsidian configs
└── .skills/        # AI Skills and automation workflows
```

## Recommended Workflow

### Obsidian

1. Install [Obsidian](https://obsidian.md/)
2. Open this repository as an Obsidian vault
3. Browse and manage notes with backlinks, graph view, and custom snippets

### Codex Skills

1. Open this repository in Codex Desktop or VS Code with Codex
2. Use the Skills in `.skills/` or create new Skills for custom workflows
3. Ask the agent to use a specific Skill when working with notes
